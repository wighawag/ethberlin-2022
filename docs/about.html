About The Bleep Machine

What if you could let the EVM sing?

The Bleep Machine allows you to create algorithmic music using EVM bytecode. Its main inspiration are the bytebeats. The bytebeats (re)discovers the possibility of generating interesting musical sounds using simple mathematical formula. This was fascinating to discover the possibilities. You can found out more about this topic here

The Bleep Machine reimplements this system completely in solidity with the constraint of the ethereum virtual machine (see yellow paper by Dr Gavin wood)

This lets anyone submit bytecode to generate music. You can see a basic example of such bytecode here. And since the whole wav file is generated by the contract it thus rely on zero infrastructure for listener, except the access to a browser with web3 access. Zero Ipfs. Zero backend.
